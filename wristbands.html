Single event tickets

Some events will have tickets available on the door. Tickets are available on a first-come-first-served basis, and are based on the number of people an event can host safely. If an event reaches capacity, wristband holders will take priority. More information will be available soon.

Guest tickets

Event tickets are available on the door to guests accompanied by a Central student. Not all events will have tickets available on the door. If an event reaches capacity, wristband holders will take priority.
